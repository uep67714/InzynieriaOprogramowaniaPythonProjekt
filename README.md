Nazwa Projektu: System Zarządzania Biblioteką
Opis Projektu
Projekt realizowany w ramach zajęć z Inżynierii Oprogramowania. Celem jest stworzenie systemu wspierającego proces wypożyczania książek, zarządzania zbiorami oraz bazą czytelników.
Funkcjonalności
•	Dodawanie nowych książek, autorów i czytelników do bazy danych.
•	Obsługa procesu wypożyczeń z walidacją dostępności egzemplarzy.
•	Wyszukiwanie wypożyczeń oraz obsługa zwrotów.
•	Weryfikacja ograniczeń (np. zakaz wypożyczenia dwóch egzemplarzy tej samej książki przez jednego czytelnika).
Struktura Działania
System opiera się na architekturze modularnej, gdzie poszczególne klasy odpowiadają za encje systemu (Książka, Czytelnik, Wypożyczenie). Procesy walidacji są zintegrowane bezpośrednio w logice wypożyczania.
